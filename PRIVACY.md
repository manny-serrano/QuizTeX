# Privacy Policy

Quizlet LaTeX Renderer does not collect, store, or transmit personal data.

## What the extension does

- Runs only on Quizlet pages (`https://*.quizlet.com/*`).
- Detects raw LaTeX text already present on the page.
- Converts that text into rendered math using MathJax in your browser.

## Data handling

- No analytics, tracking, or telemetry.
- No remote API calls made by the extension.
- No cookies are set by the extension.
- No user account data is read or stored outside the current page context.

## Permissions rationale

- Host permission for Quizlet is required so the extension can run where Quizlet content appears.
- Frame matching is enabled to support Quizlet game modes that render content in related frames.

If this changes in future versions, this policy will be updated before release.
